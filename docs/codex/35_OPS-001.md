Use together with `00_SHARED_CONTEXT.md`. Implement only this task.

```text
Implement a reusable game reboot/recovery workflow. Gracefully force-stop the configured game package through the emulator adapter, verify it stopped, launch the configured activity/package, wait through Android/game startup phases, handle only known safe popups, and verify a known home scene. Add tests for normal restart, ADB offline, package launch failure, blank first screenshot, wrong activity, and emulator-reboot escalation.
```
