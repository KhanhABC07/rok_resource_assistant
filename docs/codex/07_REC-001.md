# REC-001 — Watchdog and recovery ladder

Use together with `00_SHARED_CONTEXT.md`. Implement only this task.

## Codex Prompt

```text
Implement a watchdog that monitors emulator process, ADB connectivity, Android boot readiness, game activity, first screenshot, known scene, and same-screen timeout. Build a bounded recovery ladder: reconnect ADB, send Back, normalize to Home, relaunch game, restart emulator, then open an incident and circuit-break the instance. Make all phase timeouts configurable and persist recovery attempts. Add failure-injection tests for ADB offline, game crash, wrong activity, blank screenshot, ANR-like popup, and repeated same screen.
```
