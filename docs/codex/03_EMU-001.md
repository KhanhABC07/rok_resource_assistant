# EMU-001 — Emulator provider abstraction and command builder

Use together with `00_SHARED_CONTEXT.md`. Implement only this task.

## Codex Prompt

```text
Replace path construction based on host `pathlib.Path` semantics with a Windows-safe command builder. Define an EmulatorProvider port and a production MEmu adapter supporting discover, start, stop, force-stop game package, launch game activity, reboot, ADB connect/disconnect, screenshot, tap, swipe, keyevent, and health checks. Return typed command results with exit code, stdout, stderr, duration, and error category. Keep the current MEmu APIs backward-compatible. Add tests that assert exact Windows command lines on non-Windows CI and cover timeout, missing executable, malformed list output, ADB offline, and screenshot cleanup.
```
