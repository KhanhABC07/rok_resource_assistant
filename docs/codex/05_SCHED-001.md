# SCHED-001 — Job scheduler and configurable concurrency

Use together with `00_SHARED_CONTEXT.md`. Implement only this task.

## Codex Prompt

```text
Rework scheduling around a durable job queue and a target-aware dispatcher. Implement priority ordering, cancellation, configurable global and per-profile concurrency limits, delayed retry, optional emergency-job preemption, and restart reconciliation for queued/running jobs. Before dispatch and before critical action groups, re-read the instance session and scene; abort, reschedule, or return a typed failure when state no longer matches the job target. Add tests for priority, cancellation, concurrency limits, scene/session drift, delayed retry, emergency preemption, and application restart.
```
